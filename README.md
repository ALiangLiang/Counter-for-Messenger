## 🔥 Features

- 💬 **Count**
  - Threads (Chat rooms)
  - Messages in thread
  - Characters in thread
- 📊 **Rank** all your threads on chart.
- 💾 **Backup**(Download) your messages.
- ⚙️ **Customize** your threads. (color, emoji, name...)

